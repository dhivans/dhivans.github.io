The previous version of this bundle was two parts that didn't even include a breadboard. This one is built around an actual reason to own one: a microcontroller, a cable to power and program it, and one of each basic input/output type. It's the same parts list as the [ESP32 Mood Ring build]({{ "/projects/2026-06-20-esp32-rotary-rgb-mood-ring/" | relative_url }}), plus a rocker switch for adding physical power control to whatever you build next.

**What each part is for:**

- **ESP32 dev board** — the microcontroller. Wi-Fi and Bluetooth included, even if your first project doesn't use them yet.
- **Breadboard** — where everything else plugs in.
- **USB-A to USB-C cable** — powers and programs the board. Without it, the rest of this list is decoration.
- **KY-040 rotary encoder** — a digital input with a built-in push button.
- **WS2812B LED ring** — a programmable output you can actually see working.
- **Rotary potentiometer** — an analogue input, for when you need a value rather than an on/off event.
- **Round rocker switch** — a physical power switch you can wire into a breadboard's rail instead of unplugging USB every time.

You'll also need male-to-male jumper wires to connect everything — not currently stocked here, any generic 22–24AWG set works fine.
